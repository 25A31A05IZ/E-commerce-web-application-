# ShopEase E-Commerce Web Application

A full-stack E-Commerce application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features
- User Authentication
- Product Management
- Shopping Cart
- Checkout System
- Order Tracking
- Admin Dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Installation

npm install

npm start
